Setup Guide

Follow these steps to get your own local agent running in minutes.

1Clone the Repository

git clone https://github.com/your-username/Little_Agent_Chatbot.git
cd Little_Agent_Chatbot

2Set Up Virtual Environment

python3 -m venv .venv
source .venv/bin/activate

On Windows, use `.venv\Scripts\activate`

3Install Dependencies

pip install langchain langchain-community langchain-core langchain-ollama chromadb sentence-transformers pypdf python-dotenv unstructured[pdf] tiktoken gradio

4Download Models via Ollama

# Pull a language model (e.g., qwen2:4b)
ollama pull qwen2:4b

# Pull the model for embeddings
ollama pull nomic-embed-text

5Run the Application

Ensure Ollama is running, add your PDF files to a `data` folder, then launch the app:

python app.py

Navigate to the local URL provided by Gradio (e.g., http://127.0.0.1:7860).
